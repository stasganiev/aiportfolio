## 🎯 Функционал

### ✅ Модальное окно
```
┌─────────────────────────────────────────┐
│  ✖                                      │
│                                         │
│         Get In Touch                    │
│    Let's discuss your project           │
│                                         │
│  Name *          Email *                │
│  [________]      [________]             │
│                                         │
│  Company         Subject *              │
│  [________]      [▼______]              │
│                                         │
│  Message *                              │
│  [__________________________]           │
│  [__________________________]           │
│                                         │
│         [ Send Message ]                │
│                                         │
└─────────────────────────────────────────┘
```

- Открывается по клику на "Contact Me"
- Закрывается: крестик / ESC / клик на фон
- Backdrop blur эффект
- Блокировка скролла страницы

### ✅ Поля формы

| Поле     | Тип       | Обязательное | Валидация                    |
|----------|-----------|--------------|------------------------------|
| Name     | Text      | Да ✓         | Минимум 2 символа            |
| Email    | Email     | Да ✓         | Валидный email формат        |
| Company  | Text      | Нет          | -                            |
| Subject  | Select    | Да ✓         | Выбор из списка              |
| Message  | Textarea  | Да ✓         | Минимум 10 символов          |

**Subject опции:**
- Collaboration
- Job Offer
- Question
- Other

### ✅ Валидация

```
❌ Ошибка                    ✅ Успех
┌──────────────┐            ┌──────────────┐
│ Name         │            │ Name         │
│ [__________] │            │ [John Doe__] │
│ ⚠ Please...  │            │              │
└──────────────┘            └──────────────┘
  Красная рамка               Нормальная рамка
  Текст ошибки                Без ошибок
```

**Типы валидации:**
- Real-time (при вводе)
- On blur (при потере фокуса)
- On submit (перед отправкой)

### ✅ Отправка

```
1. User fills form
   ↓
2. Validation check
   ↓
3. Show loading spinner
   ↓
4. Send via EmailJS
   ↓
5. Hide form
   ↓
6. Show success message
   ↓
7. Trigger confetti 🎊
```

### ✅ Success состояние

```
1️⃣  Register at emailjs.com
    ↓
2️⃣  Add Email Service (Gmail)
    ↓ Copy Service ID
3️⃣  Create Template
    ↓ Copy Template ID
4️⃣  Get Public Key
    ↓
5️⃣  Update js/contact.js
    ↓
6️⃣  Test! 🚀
```
