# Images Directory

Place your images here:

## Recommended Images

1. **logo16.png** - Favicon (16x16px)
2. **profile.jpg** - Your profile photo (300x300px minimum)
3. **project screenshots** - Screenshots of your projects

## Tips

- Use optimized images (compress with tools like TinyPNG)
- Recommended formats: PNG for logos, JPG for photos, WebP for best compression
- Keep file sizes under 500KB for good performance

## Current Status

The site uses SVG placeholders and emoji for visuals. You can:
- Replace emoji with actual project screenshots
- Add your profile photo in the About section
- Add custom favicon
