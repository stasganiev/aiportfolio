## 🎯 Реализованный функционал

### 1. ✨ Parallax Эффект

```
Hero Section:
┌─────────────────────────────────┐
│  Text moves slower (0.3x)       │ ← Медленнее
│                                 │
│  [Image moves even slower]      │ ← Еще медленнее
│         (0.5x)                  │
└─────────────────────────────────┘

Создает эффект глубины при скролле
```

**Характеристики:**
- Throttled с requestAnimationFrame
- Отключается на mobile (<768px)
- Работает только в viewport Hero
- Плавная cubic-bezier анимация

---

```
Before hover:     On hover:

│ Project  │  →  │ Project  │
│          │     │          │
└──────────┘     └──────────┘
```

#### B. Border Morphing:
```
16px radius  →  24px radius + lift up
```

#### C. Gradient Glow:
```
Анимированная градиентная рамка
Цвета: синий → голубой → синий
Opacity: 0.3
```

**Все 3 эффекта работают одновременно!**

## 🎯 Оптимизация

### Throttling с RAF:
```javascript
let ticking = false;
window.addEventListener('scroll', () => {
  if (!ticking) {
    requestAnimationFrame(() => {
      updateAnimation();
      ticking = false;
    });
    ticking = true;
  }
});
```

### Mobile Detection:
```javascript
if (window.innerWidth < 768) {
  // Отключить тяжелые анимации
  disableParallax();
  hideParticles();
}
```

### GPU Acceleration:
```css
.hero-text {
  transform: translateY(var(--parallax));

}
```
